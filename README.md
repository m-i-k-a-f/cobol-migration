# COBOLマイグレーション ハンズオン

IBM Bob × COBOLマイグレーション体験ハンズオンの資料リポジトリです。

## リポジトリのダウンロード手順

### 方法1: git clone（推奨）

```bash
git clone https://github.com/m-i-k-a-f/cobol-migration.git
```

クローン後、フォルダに移動します。

```bash
cd cobol-migration
```

### 方法2: ZIPダウンロード

1. [https://github.com/m-i-k-a-f/cobol-migration](https://github.com/m-i-k-a-f/cobol-migration) を開く
2. 右上の緑色の **`<> Code`** ボタンをクリック
3. **`Download ZIP`** を選択してダウンロード
4. ZIPを展開する
