# 株式注文フロー監視ダッシュボード 実装プラン

**仕様書：** `cobol-files/order-dashboard-spec.md`
**出力ファイル：** `workshops/jip-cobol-migration/order-dashboard.html`
**目標：** 単一HTMLファイル・外部依存なし・SVG手書きグラフ・ダークテーマ

---

## トップレベルの方針

- グラフはすべて **SVG + JavaScript（DOM生成）** で自前実装。CDN・外部URL禁止。
- ダミーデータはJavaScriptオブジェクトとして固定定義（時系列は配列固定値、明細は擬似ランダム生成）。
- レイアウトは **CSS Grid / Flexbox** で1280px固定。
- 数値KPIのカウントアップは `requestAnimationFrame` で実装。

## Sub-Task 4：SVGパイチャート（注文ステータス分布）

**Intent:** 5ステータスの件数をSVGパイチャートとして描画する。CDN不使用・手計算。

**Expected Outcomes:**
- ステータス別に色分けされたパイチャートが表示される
- 各スライスに件数と%のラベルが表示される
- 凡例（ステータス名・色・件数）がチャートの右または下に表示される

**Todo List:**
1. 5色のカラーパレットを定義（約定済：`#00d4ff`、受付中：`#4488ff`、一部約定：`#ffcc00`、取消済：`#888888`、エラー：`#ff4444`）
2. 件数配列から角度を計算する関数 `calcSlices(data)` を実装
   - 合計が1件のときゼロ除算にならないよう保護
   - 全件が同一ステータスの場合（360度になるケース）のパスバグ対策：`endAngle - 0.001` を適用
3. 角度から `<path d="M...A...Z">` のSVGパス文字列を生成する関数 `slicePath(cx, cy, r, startAngle, endAngle)` を実装
4. SVG要素をJavaScriptで生成し `#pie-area` に挿入
5. 各スライスにラベル（件数 + %）をSVGテキストで描画
6. 凡例HTMLをパイチャートの下に生成

**Relevant Context:**
- 仕様書 §2-2「注文ステータスのパイチャートまたはバーチャート」「件数と割合（%）を両方表示」
- Sub-Task 1で定義した `#pie-area`
